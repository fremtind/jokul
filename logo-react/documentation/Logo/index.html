Logo

React packageStyle package

Se p GitHub

Logoen, logosymbolet og logostemplene vre er merkevareelementer vi kan bruke i Fremtinds lsninger, eller i andre lsninger der Fremtind skal vre synlig.

Du kan lese mer om hvordan vi skal bruke logo og logosymbol, under Prinsipper for bruk. Logoen og logosymbolet skal ikke st synlige sammen. Vi kan animere dem, slik at vi ser at det skifter mellom logo og logosymbol. Se eksempelet.

Vi kan bruke logostemplene der det er behov, for vise at Fremtind er en del av budskapet. Stemplene skal ikke brukes p samme mte som logoen eller logosymbolet.

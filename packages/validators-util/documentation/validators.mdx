<Ingress>Denne pakken innholder validering støttefunksjoner.</Ingress>

## Bruk

Disse validatorene kan brukes der du trenger de,
men eksempelet under viser hvordan du kan bruke disse i input som håndterer registreringsnummer for moped.

```tsx
import { isValidRegistreringsnummer } from '@fremtind/jkl-validators-util';

...
{
    required: 'Du må skrive inn et registreringsnummer',
    validate: {
        isValidRegnummer: (value: string) =>
            isValidRegistreringsnummer(value, { vehicle: 'moped' }) ||
            'Registreringsnummer må bestå av to bokstaver og fire sifre',
    },
}
...
```

## Vi har disse validatorene:

-   isValidEpost
-   isValidTelefonnummer
-   isValidRegistreringsnummer
-   isValidChassisnummer
-   isValidDogId
-   isExactLength
-   isInteger
-   isValidFodselsnummer
-   isValidOrganisasjonsnummer
-   isValidName
-   hasMinimumWords
-   hasNoIllegalCharacters

**Tips:** om du bruker [React-pakken](../table-react/) trenger du ikke installere denne pakken direkte.

1. `yarn add @fremtind/jkl-table` eller `npm i @fremtind/jkl-table`.
2. Import√©r stil-pakken i prosjektet ditt.

```js

```


**Tips:** om du bruker [React-pakken](../file-input-react/) trenger du ikke installere denne pakken direkte.

1. `npm i @fremtind/jkl-file-input`.
2. Import√©r stil-pakken i prosjektet ditt.

```js

```

```scss

```

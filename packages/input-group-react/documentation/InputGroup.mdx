## Når bruker vi InputGroup?

Annet enn når du importerer CSS skal du vanligvis ikke trenge forholde deg til denne komponenten. InputGroup er mest tiltenkt intern bruk i Jøkul.

Vi bruker den først og fremst for å gjenbruke kode, særlig rundt plassering av [Tooltip](/komponenter/tooltip) inline etter skjemafeltets label.

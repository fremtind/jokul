## FieldGroup

<ComponentExample
    title="FieldGroup"
    component={CheckboxExample}
    knobs={checkboxExampleKnobs}
    codeExample={checkboxExampleCode}
/>

FieldGroup er en komponent som brukes for å gruppere skjemafelt under en felles label, med mulighet for tooltip og hjelpetekst.

Der du sannsynligvis vil støte på denne er i forbindelse med en liste med [Checkbox](/komponenter/checkbox), men den kan brukes med alle skjemafelt hvor du ønsker en felles label.

## InputGroup

<ComponentExample
    title="InputGroup"
    component={InputGroupExample}
    codeExample={inputGroupExampleCode}
    knobs={inputGroupExampleKnobs}
/>

InputGroup koordinerer plassering av Label, [SupportLabel](#supportlabel) og [Tooltip](/komponenter/tooltip) i forhold til skjemafeltet.

Annet enn når du importerer CSS skal du vanligvis ikke trenge forholde deg til denne komponenten. InputGroup er mest tiltenkt intern bruk i Jøkul.

## SupportLabel

<ComponentExample
    title="SupportLabel"
    component={SingleCheckboxExample}
    knobs={{
        boolProps: [
            { prop: "Med hjelpetekst", defaultValue: true },
            "Med feil",
        ],
    }}
    codeExample={singleCheckboxExampleCode}
/>

SupportLabel er en komponent som brukes for å vise hjelpetekst og feilmeldinger. Som oftest vil du bruke denne via [InputGroup](#inputgroup), men det finnes unntak, slik som enkeltstående [Checkbox](/komponenter/checkbox#enkeltstående).

## Label

Label er tilknyttet et skjemafelt via en InputGroup, eller flere skjemafelter via en FieldGroup.

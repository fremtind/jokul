## Bruk

Denne hooken baserer seg på en eventListener, og den bør derfor implementeres så høyt opp i applikasjonen din som mulig for å unngå å lage for mange listeners. Bruk gjerne en React Context heller enn å bruke denne hooken flere steder i løsningen din.

`useBrowserPreferences` gir tilgang til CSS Media Queries, og kan utvides til å hente [alle verdiene som finnes der](https://developer.mozilla.org/en-US/docs/Web/CSS/@media#media_features). For øyeblikket støttes `prefers-reduced-motion` og `prefers-color-scheme`.

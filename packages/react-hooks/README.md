## Bruk av React-pakken

### Installasjon

1. Installér pakken med `yarn add @fremtind/jkl-react-hooks` eller `npm i @fremtind/jkl-react-hooks`.

2. Importer React-pakken:

```js
import { useIntersectionObserver } from "@fremtind/jkl-react-hooks";
```

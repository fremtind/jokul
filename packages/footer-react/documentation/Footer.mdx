## Når bruker vi Footer?

Footer brukes når det kreves tilleggsinformasjon om opphavsrett, kontaktinformasjon, lenker til andre sider eller lignende.

Når det gjelder innrammede applikasjoner, skal footeren til den respektive leverandøren velges.

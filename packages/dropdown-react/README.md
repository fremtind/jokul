## Bruk av React-komponenten

### Installasjon

1. Installér pakken med `yarn add @fremtind/jkl-dropdown-react` eller `npm i @fremtind/jkl-dropdown-react`. Stil-pakken blir automatisk installert som en avhengighet.
2. Importér _både_ React-pakken og stil-pakken i prosjektet ditt:

### Bruk

```js
import { Dropdown } from "@fremtind/jkl-dropdown-react";
import "@fremtind/jkl-dropdown/dropdown.min.css";
```
